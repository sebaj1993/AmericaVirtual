Para desplegar el Proyecto:

1) Tener instalado SQLServerServerExpress(Version mayor a 13.0.16)
2) Abrir la solucion con VisualStudio(Version mayor a 2015)
3) Setear Proyecto "WebChallenge" para iniciar
4) Iniciar Proyecto

Tecnologias Utilizidas:

- SqlServerExpress
- NetFramework 4.5.2
- ASP.Net 4
- WCF
- Bootstrap 3
- Entity Framework 6
- JQuery
- Api de openweathermap (https://openweathermap.org/api)
